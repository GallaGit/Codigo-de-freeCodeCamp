## 1. Diseño (Layout)

El **diseño** se enfoca principalmente en la **colocación de los elementos** dentro de una página: dónde va el encabezado, la navegación, el contenido principal, las tarjetas de producto, el pie de página, etc.

> 📁 Ejemplo práctico: `termino-comun-Diseño/` (maqueta de una tienda "ShopMate" con header, hero, grid de productos y footer).

---

## 2. Composición

La **composición** es el arte de disponer elementos para crear un diseño armonioso. Determina cómo elementos como imágenes, texto y figuras se relacionan entre sí y contribuyen al diseño de manera artística.

Mientras que el **diseño** se enfoca en la colocación, la **composición** también considera el **impacto artístico** que esa colocación tendrá en el diseño general.

- Está estrechamente relacionada con el concepto de **balance**.
- El **balance** es cómo se distribuye el peso visual dentro de una composición.
- Se busca el equilibrio mediante disposiciones **simétricas** o **asimétricas**.
- Un diseño equilibrado se siente **armonioso**.

> 📁 Ejemplo práctico: `termino-comun-Composicion/`

---

## 3. Jerarquía

La **jerarquía** establece el **orden de importancia** de los elementos en un diseño. Se trata de asegurar que la información más importante se note primero.

Se puede implementar una jerarquía visual usando:

- Tamaño
- Color
- Contraste
- Alineación
- Espacio en blanco
- Tipografía

> 📁 Ejemplo práctico: `termino-comun-Jerarquia/`

---

## 4. Contraste

El **contraste** sirve para crear distinciones claras entre elementos y **guiar la atención** del usuario hacia lo que se quiere enfatizar.

Se puede lograr mediante variaciones de:

- Color
- Tamaño
- Figura
- Textura
- Cualquier otra característica visual

Un contraste fuerte también mejora la **legibilidad**.

> 📁 Ejemplo práctico: `termino-comun-Contraste/`

---

## 5. Alineación

La **alineación** es cómo se colocan los elementos **en relación entre sí**. Usarla correctamente hace que el diseño se vea **limpio y organizado**.

Los diseñadores crean orden visual alineando elementos a lo largo de:

- Líneas imaginarias
- Bordes
- Un punto central

En CSS se practica mucho con `text-align`:

```css
.left   { text-align: left; }
.center { text-align: center; }
.right  { text-align: right; }
```

> 📁 Ejemplo práctico: `termino-comun-Alineacion/`

---

## 6. Espacio en blanco

El **espacio en blanco** (también llamado **"espacio negativo"**) es el espacio vacío en un diseño; el área que rodea a los elementos.

- No tiene que ser necesariamente de color blanco: puede ser de **cualquier color o textura**.
- Su propósito es **mejorar la legibilidad** y **realzar la jerarquía visual**.

> 📁 Ejemplo práctico: `termino-comun-Espacio-en-Blanco/`
